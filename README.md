# Telegram Username Scraper

Python script that collects usernames from Telegram groups.

## Features

- Extracts usernames from Telegram groups
- Saves data to CSV
- Uses Telegram API
- Fast and lightweight

## Technologies

- Python
- Telethon
- Pandas

## Example Output

username, id
user1,123456
user2,654321

## Installation

pip install -r requirements.txt

## Usage

python scraper.py
